Getting started

Welcome to the Heroku Userbot documentation! This guide will help you get started quickly.

Installation

To install, follow these steps:

  1. Clone the repository: 
    git clone https://github.com/coddrago/Heroku
  2. Go to the folder: 
    cd Heroku
  3. Install the dependencies: 
    pip install -r requirements.txt
The Council: It is recommended to use a Python virtual environment.

The first launch

To launch the bot, use the python3 -m heroku.

If you encounter any problems, check the FAQ.
